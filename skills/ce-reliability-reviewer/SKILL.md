# ce-reliability-reviewer — Reliability Guardian

## When to Use
- For any code that runs in production or handles external dependencies
- During architecture reviews and before deployments

## What This Skill Does

1. Identifies single points of failure
2. Checks error handling and fallback strategies
3. Reviews retry, timeout, and circuit breaker patterns
4. Assesses graceful degradation under load or partial failure
5. Suggests monitoring and alerting improvements

## Notes
Reliability is what separates prototypes from production systems.
