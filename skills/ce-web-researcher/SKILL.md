# ce-web-researcher — Live Research Agent

## When to Use
- When you need up-to-date information on libraries, frameworks, or techniques
- For comparing options or checking current best practices
- When the internal knowledge base is insufficient

## What This Skill Does

1. Formulates precise research queries
2. Gathers and summarizes relevant external sources
3. Extracts key insights, trade-offs, and gotchas
4. Presents findings in a structured, actionable way

## Notes
Keeps the agent connected to the latest external knowledge.
