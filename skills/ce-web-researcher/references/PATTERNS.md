# ce-web-researcher — Effective Patterns

## Good Research Queries
- Specific + contextual ("Next.js 15 server actions best practices 2026")
- Include constraints ("rate limiting strategies for high-traffic APIs")
- Ask for comparisons ("Redis vs Dragonfly for caching layer")

## Output Structure
- Key findings summary
- Trade-offs table
- Recommended approach with justification
- Sources / links for further reading

## When to Use
- Before making significant technical decisions
- When internal knowledge is insufficient
- To validate assumptions with current information
