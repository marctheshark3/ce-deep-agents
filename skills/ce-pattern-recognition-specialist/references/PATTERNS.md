# ce-pattern-recognition-specialist — Key Patterns

## What to Look For
- Repeated code blocks across files
- Inconsistent implementations of similar logic
- Opportunities to extract shared utilities
- Anti-patterns that have spread

## Actions
- Suggest consolidation into shared modules
- Recommend consistent patterns across the codebase
- Identify when duplication is actually acceptable (e.g., for isolation)

## Value
Helps keep the codebase DRY and consistent as it grows.
