# ce-pattern-recognition-specialist — Pattern Analyst

## When to Use
- During larger refactors or when reviewing significant portions of a codebase
- When you suspect duplication or inconsistent patterns

## What This Skill Does

1. Scans for repeated code patterns across files
2. Identifies both beneficial patterns and anti-patterns
3. Suggests extraction into shared utilities or better abstractions
4. Helps enforce consistency across the project

## Notes
Great for keeping a growing codebase clean and DRY.
