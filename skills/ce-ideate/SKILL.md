# ce-ideate — Idea Generation & Evaluation

## When to Use
- When facing an open-ended problem or feature request
- Before jumping into brainstorming or planning
- When you want creative but grounded options

## What This Skill Does

1. Generates 5–8 diverse solution ideas
2. Evaluates each against goals, constraints, and risks
3. Ranks them and recommends the top 2–3
4. Hands off the best idea(s) to `ce-brainstorm`

## Notes
Helps avoid the "first idea bias" and surfaces better approaches.
