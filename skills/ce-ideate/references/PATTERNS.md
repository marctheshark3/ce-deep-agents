# ce-ideate — Idea Generation Patterns

## Effective Approach
1. Generate diverse options (5–8 ideas)
2. Evaluate against goals, constraints, and risks
3. Rank and recommend top 2–3
4. Hand off best idea(s) to `ce-brainstorm`

## Anti-Patterns
- Jumping to the first idea
- Not considering trade-offs
- Ignoring constraints early

## Value
Helps avoid "first idea bias" and surfaces better solutions.
