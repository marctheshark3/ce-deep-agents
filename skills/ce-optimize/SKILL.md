# ce-optimize — Iterative Optimization Loop

## When to Use
- When you want to improve an existing feature (speed, clarity, conversion, etc.)
- For A/B testing ideas or prompt engineering

## What This Skill Does

1. Defines clear success metrics
2. Runs multiple parallel experiments/variants
3. Measures results objectively
4. Uses LLM-as-judge + human review for quality
5. Selects winner and compounds the winning pattern

## Notes
Powerful for continuous improvement after a feature ships.
