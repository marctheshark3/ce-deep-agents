# ce-review — Related Skills

## Skills You Should Consider Using With ce-review

### Before Review
- `ce-work` — Make sure implementation is complete
- `ce-simplify-code` — Run first if the code feels bloated
- `ce-plan` — Review the original plan for alignment

### During / After Review
- `ce-compound` — Always compound after a thorough review
- `ce-resolve-pr-feedback` — If this review came from a PR
- `ce-learnings-researcher` — Check if similar reviews were done before

### Specialized Reviewers (Used Internally by ce-review)
- `ce-code-simplicity-reviewer`
- `ce-security-reviewer`
- `ce-correctness-reviewer`
- `ce-performance-reviewer`
- `ce-architecture-strategist`
- `ce-adversarial-reviewer`
- `ce-testing-reviewer`

## Recommended Workflow
1. `ce-work` → 2. `ce-simplify-code` → 3. `ce-review` → 4. `ce-compound`
