# ce-setup — Project Bootstrap Checklist

## What ce-setup Creates
- Standard directory structure (`skills/`, `docs/`, `todos/`)
- `AGENTS.md` with core principles
- `STRATEGY.md` (if missing)
- Basic project conventions

## After Running ce-setup
- [ ] Verify all directories exist
- [ ] Read `AGENTS.md` and `STRATEGY.md`
- [ ] Run `ce-brainstorm` or `ce-lfg` on your first task
- [ ] Consider adding project-specific skills to `skills/`

## Recommended Next Steps
1. Add any project-specific patterns to `AGENTS.md`
2. Run `ce-strategy` if the high-level direction isn't clear
3. Start using `ce-lfg` for your first feature
