# ce-repo-research-analyst — Codebase Intelligence

## When to Use
- When starting work in an unfamiliar part of the codebase
- During planning to understand existing patterns and constraints

## What This Skill Does

1. Maps key directories and their responsibilities
2. Identifies established conventions and patterns
3. Surfaces historical decisions that affect new work
4. Helps new tasks align with the existing architecture

## Notes
Reduces context-switching cost and helps maintain consistency.
