# ce-work Execution Checklist

## Before Starting Work
- [ ] Clear plan exists and is approved
- [ ] All dependencies and context gathered
- [ ] Risks identified

## During Execution
- [ ] Work in small, verifiable chunks
- [ ] Run relevant tests after each chunk
- [ ] Self-review changes before moving on
- [ ] Update progress tracker regularly
- [ ] Pause for human input on risky steps

## After Major Sections
- [ ] Run broader review (`ce-review` or specific reviewers)
- [ ] Compound learnings if significant progress was made
- [ ] Update `AGENTS.md` if new patterns emerged

## Completion Criteria
- All planned items done or explicitly deferred
- Tests passing
- Code reviewed (at minimum by self + simplicity reviewer)
- Ready for final `ce-compound`
