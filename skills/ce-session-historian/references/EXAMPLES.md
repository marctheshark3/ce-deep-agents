# ce-session-historian Examples

## When This Skill is Most Valuable
- Debugging issues that may have been investigated before
- Understanding why certain decisions were made in the past
- Avoiding repeating work that was already done in previous sessions

## Example Output
"Found relevant past session from 2026-05-20 where similar auth issue was investigated. Key finding: rate limiting was added in PR #87. Recommendation: Reuse that pattern."

## Best Practice
Use early in any task that might relate to previous work. The compounding effect is strongest when we build on institutional session memory.
