# ce-session-historian — Session Memory Retriever

## When to Use
- When the current task might relate to previous work
- To avoid repeating investigations that were already done

## What This Skill Does

1. Searches historical session logs for relevant context
2. Surfaces past debugging sessions, design discussions, or decisions
3. Helps maintain continuity across multiple agent sessions

## Notes
Especially powerful when combined with persistent memory in Deep Agents.
