# ce-strategy — Product & Technical Strategy

## When to Use
- At project start or major pivot points
- Periodically to keep strategy fresh
- Before large initiatives

## What This Skill Does

1. Interviews the user (or reads existing context) about vision
2. Defines target users/personas
3. Sets success metrics (north star + supporting)
4. Documents key constraints and non-goals
5. Creates or updates STRATEGY.md as the single source of truth

## Notes
Everything else (plans, reviews, compounds) should align with this document.
