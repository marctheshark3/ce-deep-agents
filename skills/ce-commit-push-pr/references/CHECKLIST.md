# ce-commit-push-pr Checklist

## Before Running
- [ ] Code is complete and reviewed
- [ ] Tests are passing
- [ ] `ce-compound` has been run (recommended)

## What This Skill Does
1. Creates clean commit (or uses existing)
2. Pushes branch
3. Creates or updates Pull Request
4. Generates rich PR description

## Best Practices
- Run after `ce-review` for best results
- Use for any work that needs human review
- Always compound before opening PR when possible
