# ce-maintainability-reviewer — Maintainability Guardian

## When to Use
- During code review for any non-trivial change
- Periodically on larger refactors

## What This Skill Does

1. Measures coupling and cohesion
2. Identifies overly complex functions or classes
3. Flags dead code and unused imports
4. Checks naming consistency and clarity
5. Suggests refactoring opportunities that improve long-term health

## Notes
This reviewer protects the codebase from gradual decay.
