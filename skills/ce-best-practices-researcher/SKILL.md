# ce-best-practices-researcher — External Knowledge Gatherer

## When to Use
- During planning for new features or refactors
- When implementing something the team hasn't done before
- Before making significant architectural decisions

## What This Skill Does

1. Searches for established best practices in the specific domain
2. Finds real-world examples and case studies
3. Summarizes pros/cons of different approaches
4. Recommends the most suitable patterns for the current context
5. Cites sources so decisions are grounded

## Notes
This skill helps avoid reinventing wheels and making common mistakes.
