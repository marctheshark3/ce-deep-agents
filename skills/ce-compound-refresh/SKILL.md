# ce-compound-refresh — Learning Audit

## When to Use
- Every 1-2 weeks or after many new learnings have been added
- When the agent feels knowledge is getting stale

## What This Skill Does

1. Reviews recent solutions and patterns
2. Checks if they are still accurate and relevant
3. Updates or archives outdated ones
4. Merges similar learnings
5. Keeps the knowledge base clean and high-signal

## Notes
Prevents the compounding system from becoming noisy over time.
