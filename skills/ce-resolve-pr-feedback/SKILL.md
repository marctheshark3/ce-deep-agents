# ce-resolve-pr-feedback — PR Feedback Resolver

## When to Use
- After receiving review comments on a PR
- To close the loop quickly and cleanly

## What This Skill Does

1. Parses all review comments
2. Prioritizes and groups related feedback
3. Implements fixes in parallel where safe
4. Updates tests and documentation
5. Marks comments as resolved with clear explanations

## Notes
Speeds up the PR review cycle significantly.
